# Planet Texture Download Instructions

## Option 1: Three.js Repository (Easiest - Already Downloaded)
These textures are already in this folder from three.js examples:
- ✅ sun.jpg
- ✅ earth.jpg  
- ✅ moon.jpg (use for Mercury)

## Option 2: Solar System Scope (High Quality - Recommended)
Visit: https://www.solarsystemscope.com/textures/

Download these textures (2K resolution):
1. **sun.jpg** - 2k_sun.jpg
2. **mercury.jpg** - 2k_mercury.jpg
3. **venus.jpg** - 2k_venus_surface.jpg
4. **earth.jpg** - 2k_earth_daymap.jpg
5. **mars.jpg** - 2k_mars.jpg
6. **jupiter.jpg** - 2k_jupiter.jpg
7. **saturn.jpg** - 2k_saturn.jpg
8. **uranus.jpg** - 2k_uranus.jpg
9. **neptune.jpg** - 2k_neptune.jpg

Save them to this folder: `public/textures/`

## Option 3: NASA's Official Resources
Visit: https://science.nasa.gov/resource/solar-system-and-beyond-poster-set/

Download high-resolution images for each planet and save as:
- sun.jpg, mercury.jpg, venus.jpg, earth.jpg, mars.jpg, jupiter.jpg, saturn.jpg, uranus.jpg, neptune.jpg
