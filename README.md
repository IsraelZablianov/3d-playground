# 3D Particle Gestures

A real-time interactive 3D particle system using Three.js and MediaPipe Hand Tracking.

## Features

- **Hand Tracking**: Control particles with your hands via webcam.
  - **Two Hands**: Move hands apart/together to scale and expand the particles.
  - **Swipe**: Swipe hand left/right to rotate the object.
- **Templates**: Choose from Heart, Flower, Saturn, Buddha, and Fireworks.
- **Color Control**: Real-time color adjustment.
- **Particle System**: 8000 particles morphing between shapes.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to the local URL (usually `http://localhost:5173`).
4. Allow camera access when prompted.

## Controls

- **Scale/Expand**: Show both hands and move them apart.
- **Rotate**: Swipe one hand horizontally.
- **Morph**: Use the UI panel to select different shapes.
- **Color**: Use the UI panel to pick colors.
